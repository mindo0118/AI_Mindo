문제 정의

소상공인과 1인 창업가들은 온라인 판매를 시작할 때 코딩 및 디자인 지식의 부재와 전문 에이전시 의뢰 시 발생하는 높은 비용과 긴 제작 기간이라는 장벽에 부딪힙니다.

해결책

별도의 호스팅 설정이나 개발 지식 없이도 상품 이미지만 있으면 즉시 고품질의 랜딩 페이지를 확보할 수 있는 자동화 솔루션을 개발했습니다.

시스템 아키텍처: Input → AI Analysis (Gemini) → Asset Gen (SDXL) → Single-File HTML

핵심 기능

멀티모달 AI 기반 콘텐츠 생성

Google Gemini 2.5 Flash를 활용하여 이미지를 시각적으로 분석하고, 키워드(예: '짭짤한')를 반영한 마케팅 문구와 헤드라인을 자동 작성합니다. 이를 반응형 HTML/CSS 코드로 즉시 변환합니다.

Gemini 2.5 Flash HTML/CSS Gen

지능형 이미지 생성 및 자동화

Stability AI (SDXL) 모델을 연동하여 웹사이트 테마에 맞는 배너 이미지를 생성(Text-to-Image)하고, 원본 이미지를 웹 디자인에 맞게 보정합니다. 모든 에셋은 Base64로 통합되어 단일 파일로 배포됩니다.

Stability AI (SDXL) Asset Optimization

나의 역할 및 기여

  • Backend API 개발: FastAPI 비동기 파이프라인(이미지 분석 → 텍스트 → 코드) 로직 구현
  • Prompt Engineering: Gemini가 문법 오류 없는 HTML 구조와 매력적인 카피를 출력하도록 최적화
  • AI Integration: Stability AI 연동 및 이미지 생성 로직 구현
  • 패키징 구현: 결과물을 ZIP 및 Single-File HTML로 변환하는 기능 개발

Project Retrospective

배운 점

LLM(Gemini)이 항상 일관된 코드 형식을 반환하도록 제어하는 프롬프트 엔지니어링의 중요성을 깊이 있게 익혔습니다. 또한, FastAPI의 Async/Await 비동기 처리를 통해 무거운 AI 모델 추론 대기 시간 동안 서버 자원을 효율적으로 관리하는 방법을 체득했습니다.

개선할 점

현재는 정적 파일(HTML)로 제공되지만, 향후에는 사용자가 드래그 앤 드롭으로 직접 수정할 수 있는 에디터 기능을 추가하고 싶습니다. 또한 DB 스키마를 고도화하여 사용자 요청 이력을 관리하고, 재생성 시 이전 스타일을 기억하는 기능을 도입할 예정입니다.
